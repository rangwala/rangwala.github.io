Machine Learning for Wearables and IOTs in Biomedical Applications


As part of this effort, we collaborate with bioengineers to develop next-generation technologies for assistive livining. Examples include collaborating on development of a prosthetic arm using ultrasound imaging for transradial amputees. Using the input imaging data the objective of data mining algorithms is to develop classification algorithms relying on multi-task learning for fine hand control. This project is led by Professor Sikdar in the Bioengineering department and funded by NSF Cyber-Physical Sciences Program.

Similarly in collaboration with Professor Parth Pathak , expert in WiFi technologies and Professor Raja Kushalnagar at Gallaudet University we are working on gesture recognition algorithms.
