Project Ideas and Listings.

17 Jan 2009
Posted by rangwala

The project gives you an opportunity to explore in depth a particular topic/area of the course that interests you. The topic of the project, of course, should be related to the material covered in class, but otherwise you are free to select the specific topic. I can help you define a topic too. Talk to me if you had like that.

Possible types of projects include:

An application research project: The project demonstrates the application of some techniques discussed in class in an application domain (e.g., sequence analysis, metagenomics, fragment assembly, gene finding, structure prediction etc.). Properties, drawbacks, advantages of the used techniques are analyzed within the context of the explored application domain.

A theoretical or methodological research project: A study of different classes of models and approaches; proving either theoretically or experimentally properties of known algorithms; designing a new approach.

Any other projects than ones listed below are fine too. These are just suggestions.
I will participate in CASP 9 - so I always like to work on new methods for protein structure prediction. Hence, there is an emphasis on structural bioinformatics. Also, I would be happy to help you with your thesis plans if you feel any could be extended to the same.

Here are some project ideas:

New Sequencing Technologies Related

  1. Implement a parallel version of a resequencing type (alignment-based) assembler like SeqMap
  2. Implement a GPU/cell-based parallel version of assemblers like SeqMap
  3. Implement an assembler that is a hybrid of denovo + resequencing type. Example papers are: VELVET + SeqMap
  4. Survey a whole bunch of short read assemblers. Assess their performance on a benchmark chosen by you or by me.

Structural Bioinformatics:

  1. Write a survey paper describing all the different papers on protein-ligand interactions
  2. Develop a function prediction algorithm using profile-based kernel functions.

    A good benchmark for function prediction paper to start with a benchmark is

    Jian Qiu, Martial Hue, Asa Ben-Hur, Jean-Philippe Vert and William Stafford Noble. "A structural alignment kernel for protein structures." Bioinformatics. 23(9):1090-1098, 2007. Benchmark here here

    3. You can use the profile kernels I developed in 2005. The paper is

    Huzefa Rangwala, George Karypis. Profile-based Direct Kernels for Remote Homology Detection and Fold Recognition in BIOINFORMATICS, 21(23):4239-4247 (2005).
    and the code is here

  3. Develop an algorithm to predict some functional or structural property (residue-wise annotation). For example look at the services I provide in MONSTER and the backend software PROSAT .

    The paper to refer is available here.

  4. Extend PROSAT to accept structure-based features. Examples would be incorporation of sequence+structure features for predicting ligand-binding site residues.
  5. Develop a contact map prediction algorithm.

    Papers to refer include SVMCon . Also refer to the CASP Prediction Competition

  6. Model the loop region of Dopamine Receptor Protein - Human. Dr. Nadine Kabbani is very interested in this for therapeutic properties. The project should involve running several bioinformatic analyses to model the protein, the loop regions, and mutated proteins.
  7. Define a problem based on the definitions provided in CASP .
  8. Extend my remote homology detection work

    Huzefa Rangwala, George Karypis. Profile-based Direct Kernels for Remote Homology Detection and Fold Recognition in BIOINFORMATICS, 21(23):4239-4247 (2005)

    for developing a complete fold recognition system for any set of protein sequence.

    The open questions are

    1. Can the work be improved upon ? There are countless papers on this topic since 2005, but none has gone ahead to beat the results shown
    2. How to handles sequences that are rare or novel
    3. How to build a complete server ?

Metagenomics:

  1. Build a taxonomical classifier for bacterial sequences that may be available as short reads or contigs. See
    the work:

    TACOA -Taxonomic classification of environmental genomic fragments using a kernelized nearest neighbor approach
    Naryttza N Diaz , Lutz Krause , Alexander Goesmann , Karsten Niehaus and Tim W Nattkemper
    BMC Bioinformatics 2009, 10:56doi:10.1186/1471-2105-10-56

  2. Extend PROSAT for genome annotation in case of single genomes, metagenomes, or reads of these metagenomes

Systems Biology Related

  1. Come up with a pairwise network alignment algorithm. Check this survey for more ideas.
  2. Implement a genetic algorithm for pairwise network alignment. Here is an example of a paper on the topic.
