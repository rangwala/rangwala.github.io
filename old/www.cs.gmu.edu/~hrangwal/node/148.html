Assignment 1: Due Date 02.15.2010 (Before Class Begins)

This is a two-part assignment. The first-part is a programming assignment (100 points). The second-part (50 points) needs the following questions to be answered below:

Part II

  1. Describe two biological applications where we would use sequence alignment algorithms. Demonstrate a clear understanding of when the alignment score is important and when the final alignment is of interest. When would you use global sequence alignment versus local sequence alignment ?
  2. Discuss a strategy to modify the programs written in Part I, such that the first input sequence can have two times the number of gaps (in general) than the second input sequence. You do not have to implement it.

The due date is 02.15.2010. This assignment has to be done individually.
