HW0: Familiarity with SPIM (MIPS Simulator)

Objectives: The objective of this assignment is to get you familiar with the SPIM MIPS simulator installed on GMU's ITE "zeus" cluster or if you chose to install it on your own computers/laptops/hardware.

Grading: 0%. Performing this assignment will help you do the follow on assignments.

Due Date: There is no deliverable for this assignment. I recommend that you complete this assignment by 09.08.2010.

Introduction: SPIM is a simulator that lets you execute assembly code designed for the MIPS R2000/R3000 architecture. MIPS is a RISC architecture that has a simple instruction set, has 32 bit registers and is used for education purposes as well as in several embedded systems.

  • [Step 1]: Creating a SPIM Working environment.

    There are two options here:

    1. Please download and install SPIM and XSPIM on the platform you prefer [PC, Linux or OSX]. The latest version of SPIM may not work on OSX and PCSPIM on Windows is like XSPIM on Linux.
    2. SPIM and XSPIM both are installed for you on "zeus.ite.gmu.edu".

      Here is a FAQ guide on how-to-ssh into our ITE machines .

  • [STEP 2]: Is SPIM working for you ?

    Type spim at the terminal and you should see this below


    -bash-3.2$ spim
    SPIM Version 8.0 of January 8, 2010
    Copyright 1990-2010, James R. Larus.

    See the file README for a full copyright notice.

    Please read Getting started with XSPIM for Linux or Mac users.

    OR

    Please read Getting started with PCSpim for PC users.

    AND

    Please read Getting started with SPIM.

  • [STEP 4]: Test your first MIPS program.
    1. Load your favorite editor and type the code below for Hello World example. Save the file as hello_world.s


      #Author: Huzefa Rangwala
      #Date: 08.29.2010


      .data
      silly_str: .asciiz "Hello World, My first MIPS program\n"


      .text


      main:
      li $v0, 4
      la $a0, silly_str
      syscall
      jr $ra

    2. Run spim or xspim or PCSPIM
    3. Load your program. In spim command line type

      load "hello_world.s"

    4. Run the loaded program. In command-line type

      run

    5. Note down the output
  • [Step 5] Congratulations on setting up and running your first MIPS program. If you encounter difficulties, please email tbd@gmu.edu or rangwala@cs.gmu.edu or meet us in our office hours. We are here to help
