MIPS Programming (Part 1 - 50 points)
Maximal Knight's Tour Problem: In HW2 you implemented a knight's move. This assignment takes it a step further and tests your understanding of recursive MIPS programming. A knight's tour is a sequence of moves of a knight on a n X n chessboard such that the knight visits every square exactly once. For some chessboards (for example of size 4 x 4) it is impossible to find a valid knight tour. A maximal knight's tour is defined as the path which has the maximum number of possible moves. Your goal is to write MIPS program that takes as input "n", which is the n X n size of the chessboard under consideration and prints a valid maximal knight tour. Your implementation should have a recursive MIPS function. For your reference we show valid knight's movement and two tours on a 5x5 and a 8x8 chessboard below
