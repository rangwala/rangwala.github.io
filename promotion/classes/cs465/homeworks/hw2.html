MIPS Programming (Part 1 - 50 points)
Phone Numbers : The goal of this program is to write a set of functions that computes the number of unique possible phone numbers based on the constraints below. Your code should be as efficient as possible and should use only the MIPS basic instructions. Constraints:
  • Each valid phone number is only 7 in length. Only digits are allowed.
  • A valid number does not begin with a 0 or 1.
  • A valid number is one that can be traced by a sequence of knight movements on a phone keyboard (shown below)
    Phone Layout
    1 2 3
    4 56
    789
    *0#
    Knight Movement
    X X
    X X
    O
    X X
    X X
    Given the entry 0 we show the valid next possible entries as X. As an example, given the digit 8 the next possible digits can be 1 or 3 only.

  • A. abs $t2, $t3 # R[rd] = | R[rs] |
